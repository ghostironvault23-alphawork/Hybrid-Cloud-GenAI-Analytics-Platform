# Roadmap

## Phase 1: Local MVP

- FastAPI backend
- Local data lake
- SQLite transaction database
- Mock AI service
- Static frontend
- Sample ingestion scripts

## Phase 2: AWS Serverless

- S3 data lake
- API Gateway
- Lambda
- DynamoDB
- Cognito
- CloudWatch
- Bedrock

## Phase 3: Azure Cloud Extension

- Azure Data Lake Storage Gen2
- Azure App Service / Functions
- Cosmos DB
- Azure OpenAI
- Azure AI Search
- API Management
- Application Insights

## Phase 4: Enterprise Data and AI

- Glue crawlers / Azure Data Factory
- Athena / Synapse analytics
- QuickSight / Power BI dashboards
- Bedrock Knowledge Bases / Azure AI Search
- PII detection and AI guardrails

## Phase 5: Enterprise Scale

- ECS/EKS or AKS microservices
- Step Functions or Logic Apps workflows
- Queue-based failure handling
- PrivateLink / VPN / Direct Connect
- CI/CD with GitHub Actions
- Multi-account or landing zone strategy
